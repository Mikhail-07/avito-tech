# avito-tech

## Подготовка к запуску проекта:

Перед запуском необходимо создать в корневой папке файл .env с переменной REACT_APP_TOKEN и указать в ней token API кинопоиска

## Запуск проекта в режиме разработчика: 

### `npm run start` 

## Сборка webpack:

### `npm run build:dev` 
### `npm run build:prod`
